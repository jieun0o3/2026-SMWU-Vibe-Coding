## MODIFIED Requirements

### Requirement: 오늘 합주 몇 번째인지 알림
앱 진입 시 오늘 합주가 있다면, 오늘이 전체 합주 중 몇 번째 합주인지 배너로 알려야 한다(SHALL).

#### Scenario: 오늘 합주 당일 알림 — 모든 페이지
- **WHEN** 사용자가 오늘 합주가 있는 날 앱의 어떤 페이지에 접속하든
- **THEN** 헤더 바로 아래에 "오늘은 N번째 합주입니다! 합주 시간: HH:MM" 형태의 배너가 표시된다

#### Scenario: 오늘 합주 없는 날
- **WHEN** 오늘 합주가 없는 날 앱에 접속하면
- **THEN** 합주 알림 배너가 표시되지 않는다
