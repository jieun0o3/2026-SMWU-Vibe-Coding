## MODIFIED Requirements

### Requirement: 남은 합주 횟수 표시
시스템은 공연 전까지 남은 합주 횟수를 캘린더 상단에 표시해야 한다(SHALL).

#### Scenario: 남은 합주 횟수 표시
- **WHEN** 사용자가 캘린더 페이지에 접속하면
- **THEN** "남은 합주: N회" 형태로 아직 진행되지 않은 합주 횟수가 표시된다

#### Scenario: 합주가 모두 완료된 경우
- **WHEN** 등록된 합주가 1건 이상이고 오늘 이후 남은 합주가 0건이면
- **THEN** 배지가 녹색으로 전환되어 "모든 합주 완료 🎉" 텍스트가 표시되고, 페이지 하단에 "모든 합주를 완료했습니다! 공연 파이팅!" 메시지가 표시된다
