## ADDED Requirements

### Requirement: 합주 수동 추가
팀 대표는 자동 배분과 별개로 날짜와 시각을 직접 입력하여 합주를 1건씩 추가할 수 있어야 한다(SHALL).

#### Scenario: 수동 합주 추가 성공
- **WHEN** 팀 대표가 날짜와 시각을 입력하고 추가 버튼을 누르면
- **THEN** 해당 날짜·시각의 합주가 목록에 추가되고, 순번이 날짜 순으로 재정렬된다

#### Scenario: 날짜 미입력 시 차단
- **WHEN** 날짜를 입력하지 않은 채 추가 버튼을 누르면
- **THEN** 추가가 실행되지 않고 날짜를 입력하라는 안내 메시지가 표시된다
