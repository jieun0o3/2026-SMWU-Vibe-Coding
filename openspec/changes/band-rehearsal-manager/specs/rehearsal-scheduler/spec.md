## ADDED Requirements

### Requirement: 합주 일정 자동 생성
팀 대표는 공연 날짜와 원하는 합주 횟수를 입력하면, 시스템이 모든 멤버가 가능한 날짜 중에서 균등한 간격으로 합주 일정을 자동으로 생성해야 한다(SHALL).

#### Scenario: 합주 일정 자동 생성 성공
- **WHEN** 팀 대표가 공연 날짜, 합주 횟수, 합주 시작 시각을 입력하고 자동 배분 버튼을 누르면
- **THEN** 오늘부터 공연 전날까지 모든 멤버가 가능한 날짜 목록을 계산하여, 균등 간격으로 N개의 합주 날짜가 자동으로 배정된다

#### Scenario: 가능한 날짜 부족 시 경고
- **WHEN** 모든 멤버가 가능한 날짜 수가 요청한 합주 횟수보다 적으면
- **THEN** 확보 가능한 최대 횟수와 함께 경고 메시지가 표시되고, 가능한 횟수만큼만 배정된다

#### Scenario: 공연 날짜 미입력 시 차단
- **WHEN** 팀 대표가 공연 날짜를 입력하지 않은 채 자동 배분 버튼을 누르면
- **THEN** 자동 배분이 실행되지 않고 공연 날짜를 입력하라는 안내 메시지가 표시된다

### Requirement: 합주 일정 수동 조정
팀 대표는 자동 생성된 합주 일정을 수동으로 추가, 변경, 삭제할 수 있어야 한다(SHALL).

#### Scenario: 합주 날짜 변경
- **WHEN** 팀 대표가 기존 합주 날짜를 다른 날짜로 수정하고 저장하면
- **THEN** 해당 합주의 날짜가 변경되고, 합주 순번(N번째)은 날짜 순으로 재정렬된다

#### Scenario: 합주 삭제
- **WHEN** 팀 대표가 특정 합주를 삭제하면
- **THEN** 해당 합주가 일정에서 제거되고 남은 합주의 순번이 재정렬된다
