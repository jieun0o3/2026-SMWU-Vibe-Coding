## ADDED Requirements

### Requirement: 멤버별 피드백 작성
합주가 끝난 후 각 멤버는 해당 합주에 대한 피드백을 작성할 수 있어야 한다(SHALL).

#### Scenario: 피드백 작성 성공
- **WHEN** 멤버가 자신의 이름을 선택하고 피드백 내용을 입력한 뒤 저장하면
- **THEN** 해당 합주의 피드백 목록에 멤버 이름과 내용, 작성 시각이 저장되어 표시된다

#### Scenario: 내용 미입력 시 차단
- **WHEN** 멤버가 내용을 비운 채 저장 버튼을 누르면
- **THEN** 저장이 실행되지 않고 내용을 입력하라는 안내 메시지가 표시된다

#### Scenario: 피드백 수정
- **WHEN** 멤버가 자신이 작성한 피드백의 수정 버튼을 누르고 내용을 변경한 뒤 저장하면
- **THEN** 기존 피드백 내용이 새 내용으로 업데이트된다

### Requirement: 합주 피드백 목록 조회
모든 멤버의 피드백이 합주별로 조회 가능해야 한다(SHALL).

#### Scenario: 합주별 피드백 목록 표시
- **WHEN** 사용자가 특정 합주의 상세 페이지를 열면
- **THEN** 해당 합주에 작성된 모든 멤버의 피드백이 작성 시각 순으로 표시된다

### Requirement: 합주 녹음 파일 업로드
멤버는 합주별로 녹음 파일을 1개 이상 업로드할 수 있어야 한다(SHALL).

#### Scenario: 녹음 파일 업로드 성공
- **WHEN** 멤버가 합주 상세 페이지에서 파일 선택 후 업로드 버튼을 누르면
- **THEN** 선택한 파일(복수 가능)이 서버에 저장되고 파일 목록에 파일명이 표시된다

#### Scenario: 파일 크기 초과 시 차단
- **WHEN** 업로드하려는 단일 파일의 크기가 20MB를 초과하면
- **THEN** 업로드가 차단되고 "파일 크기는 20MB 이하여야 합니다" 안내 메시지가 표시된다

#### Scenario: 녹음 파일 다운로드
- **WHEN** 사용자가 업로드된 파일 목록에서 파일명을 클릭하면
- **THEN** 해당 파일이 다운로드된다
