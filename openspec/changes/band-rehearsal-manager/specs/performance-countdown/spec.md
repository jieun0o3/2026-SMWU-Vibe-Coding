## ADDED Requirements

### Requirement: 공연 D-day 카운트다운
시스템은 공연까지 남은 일수를 앱 상단에 항상 표시해야 한다(SHALL).

#### Scenario: D-day 카운트다운 표시
- **WHEN** 사용자가 앱의 어떤 페이지에 접속하든
- **THEN** 헤더 또는 상단 배너에 "공연까지 D-N" 형태로 남은 일수가 표시된다

#### Scenario: 공연 당일
- **WHEN** 오늘이 공연 날짜와 같은 날이면
- **THEN** "D-Day! 오늘 공연입니다" 메시지가 강조 표시된다

#### Scenario: 공연이 지난 경우
- **WHEN** 공연 날짜가 오늘보다 이전이면
- **THEN** D-day 카운트다운 대신 "공연 완료" 상태가 표시된다

### Requirement: 오늘 합주 몇 번째인지 알림
앱 진입 시 오늘 합주가 있다면, 오늘이 전체 합주 중 몇 번째 합주인지 배너로 알려야 한다(SHALL).

#### Scenario: 오늘 합주 당일 알림
- **WHEN** 사용자가 오늘 합주가 있는 날 앱에 접속하면
- **THEN** 페이지 상단에 "오늘은 N번째 합주입니다! 합주 시간: HH:MM" 형태의 배너가 표시된다

#### Scenario: 오늘 합주 없는 날
- **WHEN** 오늘 합주가 없는 날 앱에 접속하면
- **THEN** 합주 알림 배너가 표시되지 않는다
