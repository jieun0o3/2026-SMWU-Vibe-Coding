## ADDED Requirements

### Requirement: 멤버 불가 날짜 등록
멤버는 자신의 이름을 선택한 뒤, 합주에 참여할 수 없는 날짜를 하나 이상 등록할 수 있어야 한다(SHALL).

#### Scenario: 불가 날짜 등록 성공
- **WHEN** 멤버가 이름을 선택하고 날짜를 1개 이상 선택한 뒤 저장 버튼을 누르면
- **THEN** 해당 날짜들이 그 멤버의 불가 일정으로 저장되고 화면에 반영된다

#### Scenario: 날짜 미선택 시 저장 차단
- **WHEN** 멤버가 이름만 선택하고 날짜를 선택하지 않은 채 저장 버튼을 누르면
- **THEN** 저장이 실행되지 않고 날짜를 선택하라는 안내 메시지가 표시된다

### Requirement: 멤버 불가 날짜 삭제
멤버는 이미 등록한 불가 날짜를 삭제할 수 있어야 한다(SHALL).

#### Scenario: 불가 날짜 삭제 성공
- **WHEN** 멤버가 자신의 등록된 불가 날짜 중 하나를 선택하고 삭제 버튼을 누르면
- **THEN** 해당 날짜가 불가 일정에서 제거되고 목록에서 사라진다

### Requirement: 멤버 불가 날짜 조회
시스템은 멤버별 불가 날짜 목록을 조회할 수 있어야 한다(SHALL).

#### Scenario: 불가 날짜 목록 표시
- **WHEN** 멤버가 자신의 이름을 선택하면
- **THEN** 현재 등록된 불가 날짜 목록이 날짜 오름차순으로 표시된다
