## ADDED Requirements

### Requirement: 합주 일정 캘린더 표시
시스템은 월간 캘린더 형식으로 합주 일정을 시각적으로 표시해야 한다(SHALL).

#### Scenario: 합주 날짜 캘린더 표시
- **WHEN** 사용자가 캘린더 페이지에 접속하면
- **THEN** 현재 월의 캘린더가 표시되고, 합주가 있는 날짜에는 합주 표시(N번째 합주)가 나타난다

#### Scenario: 월 이동
- **WHEN** 사용자가 이전/다음 월 이동 버튼을 누르면
- **THEN** 해당 월의 캘린더로 전환되며, 해당 월에 속한 합주 일정이 표시된다

#### Scenario: 오늘 날짜 강조
- **WHEN** 캘린더가 표시될 때
- **THEN** 오늘 날짜가 시각적으로 강조(테두리 또는 배경색)되어 구별된다

### Requirement: 남은 합주 횟수 표시
시스템은 공연 전까지 남은 합주 횟수를 캘린더 상단에 표시해야 한다(SHALL).

#### Scenario: 남은 합주 횟수 표시
- **WHEN** 사용자가 캘린더 페이지에 접속하면
- **THEN** "남은 합주: N회" 형태로 아직 진행되지 않은 합주 횟수가 표시된다

#### Scenario: 합주가 모두 완료된 경우
- **WHEN** 공연일 이후 모든 합주가 종료되면
- **THEN** "남은 합주: 0회, 모든 합주 완료" 메시지가 표시된다
