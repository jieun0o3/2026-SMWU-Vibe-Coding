# Product Brief

## 1. Product Summary

이 서비스는 밴드 멤버들이 합주 일정 조율의 번거로움을 개인 일정 기반 자동 스케줄링으로 해결하도록 돕는다.

## 2. Problem

밴드는 보통 여러 명으로 구성되어 있어 모두의 일정이 맞는 날을 찾는 과정이 번거롭다. 카카오톡이나 구글 폼으로 개인 일정을 모아 수동으로 비교하는 방식은 시간이 많이 걸리고 실수가 생기기 쉽다. 공연 날짜가 정해진 상황에서 합주를 몇 번, 언제 해야 할지 계획하기도 어렵다. 합주가 끝난 뒤 피드백을 체계적으로 기록하는 수단이 없어 개선이 느리다. 합주 당일 알림이나 진척도 파악 수단이 없어 공연 준비 상황을 한눈에 보기 어렵다.

## 3. Target User

- 주요 사용자: 공연을 준비하는 소규모 밴드 멤버 (2~8인)
- 사용 상황: 합주 일정 조율, 공연 준비 현황 확인, 합주 후 피드백 기록
- 사용자의 기술 수준: 스마트폰과 웹 브라우저에 익숙한 일반 사용자
- 사용자가 기대하는 결과: 번거로운 일정 조율 없이 최적의 합주 일정이 자동으로 잡히고, 공연까지의 준비 현황을 한눈에 확인할 수 있다

## 4. Core Value

개인 불가 일정을 입력하면 공연일까지 최적의 합주 일정을 자동으로 잡아주고, 진행 현황과 피드백을 한 곳에서 관리한다.

## 5. MVP Scope

- 밴드 멤버 등록 및 개인 불가 일정 입력
- 공연 날짜와 합주 횟수를 입력하면 자동으로 합주 일정 생성
- 합주 일정 캘린더 뷰 + 남은 합주 횟수 표시
- 당일 합주 알림 및 몇 번째 합주인지 표시
- 합주별 멤버 피드백 작성 및 녹음 파일 업로드 (여러 개 가능)

## 6. Non-goals

- 로그인 / 사용자 인증
- 결제
- 실시간 협업 (동시 편집)
- 외부 캘린더 API 연동 (구글 캘린더 등)
- 다중 외부 API 연동
- 복잡한 관리자 권한
- 서버 푸시 알림

## 7. Success Criteria

- 사용자는 핵심 기능을 1분 안에 사용할 수 있다.
- 3회차 종료 시 브라우저에서 기본 흐름(멤버 등록 → 불가 일정 입력 → 합주 자동 생성 → 캘린더 확인 → 피드백 작성)을 확인할 수 있다.
- 4회차에서 핵심 흐름을 Playwright 테스트로 검증할 수 있다.
- 최종 결과물은 배포 가능한 형태여야 한다.
